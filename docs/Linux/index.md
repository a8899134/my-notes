# Linux 系统笔记
Linux 是一个开源、类 Unix 的操作系统内核，广泛用于服务器、云计算、嵌入式设备等领域。作为运维工程师，Linux 是必须掌握的核心技能。
## 🧠 为什么选择 Linux？
- **开源免费**：绝大多数发行版可免费使用，代码公开透明
- **稳定高效**：长期运行的服务器几乎不需要重启，资源占用可控
- **安全性强**：用户权限体系完善，社区安全响应迅速
- **生态丰富**：几乎所有的开源软件、中间件、容器技术都优先支持 Linux
